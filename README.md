> A tool to fix media file names

## 🛠️ Features

* Fixes media filenames with random characters.

## 📦 Installation

```bash
pip install reelname
```

## 🧪 Usage

* To view the help message, run the following command:

```bash
reelname --help
```

## 📐 Requirements

* Python >= 3.9
